# アソビュー自動入力オーバーレイ

`https://www.asoview.com/purchase/*` 向けの Chrome 拡張です。  
ページ右上に以下の2つのボタンを表示します。

- `購入者情報を入力`
- `カード情報を入力`

## 機能

- オプション画面で設定した購入者情報の自動入力
- オプション画面で設定したカード情報の自動入力
- カード入力は現在ドキュメントと到達可能な iframe を探索
- クロスオリジン iframe は `postMessage` 連携で入力を試行
- 設定保存先は `chrome.storage.local`

## 購入者情報の入力項目

- 姓 / 名
- セイ / メイ
- 電話番号（前・中・後）
- 性別
- 生年月日（年・月・日）
- 郵便番号（前・後）
- 都道府県（コード選択）
- 市区町村以下

## カード情報の入力項目

- カード番号
- 有効期限（月・年）
- CVC
- カード名義

## セットアップ

1. `chrome://extensions/` を開く
2. デベロッパーモードを有効化
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このフォルダを選択
5. 拡張機能のオプションを開いて値を保存

## 注意

- カード情報は `chrome.storage.local` に平文保存されます。
- カード入力 iframe 対応のため、拡張は全サイト権限を使用します。
- 対象ページの DOM が変わるとセレクタが一致しなくなる場合があります。
